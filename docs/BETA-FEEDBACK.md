## 5. 腾讯问卷配置指南

### 推荐问题列表（7-8题）

1. **总体满意度**
   - 类型：1-5星评分
   - 问题：你对「思维进化训练营」的总体满意度如何？

2. **课程内容实用性**
   - 类型：1-5星评分
   - 问题：课程内容对你的实际工作和生活有帮助吗？

3. **AI互动体验**
   - 类型：1-5星评分
   - 问题：与AI老师的互动体验如何？

4. **界面易用性**
   - 类型：1-5星评分
   - 问题：应用界面是否清晰易用？

5. **最喜欢的课程/功能**
   - 类型：开放题
   - 问题：你最喜欢哪节课或哪个功能？为什么？

6. **最需要改进的地方**
   - 类型：开放题
   - 问题：你觉得产品最需要改进的地方是什么？

7. **推荐意愿**
   - 类型：是/否
   - 问题：你会向朋友推荐这个产品吗？

8. **其他建议**
   - 类型：开放题
   - 问题：还有什么想对我们说的？

### 问卷创建步骤

1. 打开 [腾讯问卷](https://wj.qq.com)
2. 创建新问卷 → 选择「空白问卷」
3. 添加上述8个问题
4. 设置问卷主题为「思维进化训练营 - 内测反馈」
5. 发布问卷，复制链接替换 Profile.jsx 中的 `FEEDBACK_URL`
6. 在群内通知用户问卷已上线
