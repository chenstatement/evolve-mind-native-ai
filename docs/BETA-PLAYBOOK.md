# 内测用户手册

> 欢迎加入「思维进化训练营」种子用户内测！

---

## 1. 欢迎使用

你好！感谢你成为「思维进化训练营」的产品共创者。

这是一个面向25-45岁职场人的AI引导式思维训练课程。我们设计了10节结构化课程，每节课包含故事引入、知识讲解、互动练习和AI对话四个环节，帮助你在AI时代建立更高效的思维方式。

你的身份不是"测试小白鼠"，而是**产品共创者**。你的每一条反馈都会直接影响产品的迭代方向。

---

## 2. 快速开始

### 3步开始你的思维进化之旅：

**第1步：解锁课程**
- 打开 https://chenshushi.com（手机浏览器即可）
- 点击右下角「我的」→「课程解锁」
- 输入你的专属兑换码

**第2步：开始学习**
- 返回首页，点击第1课「成长型思维」
- 跟随AI老师的引导完成学习
- 每节课约15-20分钟

**第3步：提交反馈**
- 完成全部10课后查看结业证书
- 弹出评分问卷，花1分钟填写
- 或通过「我的」→「反馈建议」填写深度问卷

## 4. AI互动说明

### 如何与AI老师互动

课程中的AI老师不是简单的问答机器人，而是会根据你的回答调整引导方式的"思维教练"。

**提问技巧**：
- 开放式提问效果最好："我对这个概念的理解是……对吗？"
- 可以分享你的真实困惑："我在工作中经常遇到……"
- 尝试用课程中学到的框架分析实际问题

**AI配置**（可选）：
- 如果你想使用更强大的AI模型，可以在「我的」→「AI配置」中设置自己的API Key
- 支持 DeepSeek、OpenAI 等兼容OpenAI格式的API
- 不配置也没关系，内置的AI对话功能已足够使用

## 5. 反馈方式

我们设计了三种反馈渠道，你可以任选：

### 渠道1：结业评分（推荐）
- 完成全部10课后，结业证书页面会自动弹出评分弹窗
- 5星评分 + 两个开放题（最喜欢 / 最需改进）

- 深度问卷，包含满意度、课程实用性、AI体验等
- 约需3-5分钟

### 渠道3：微信群反馈
- 日常使用中遇到任何问题，直接在群里说
- 看到任何觉得不合理的交互，随时截图发群里

## 6. 常见问题

**Q：我的学习数据安全吗？**
A：所有学习数据都存储在你自己的手机浏览器中（localStorage），不会上传到任何服务器。清除浏览器缓存会导致进度丢失，建议不要在学习期间清除缓存。

**Q：我能在不同设备上同步进度吗？**
A：目前不支持跨设备同步，建议固定在一台设备上学习。未来版本会考虑加入账号系统。

**Q：兑换码只能使用一次吗？**
A：是的，每个兑换码只能使用一次，且绑定到当前设备。请妥善保存你的兑换码。

**Q：我可以把课程分享给朋友吗？**
A：完成内测后，你会收到一个正式版兑换码，可以转赠给朋友。但请不要分享内测兑换码，因为内测名额有限。

**Q：AI对话回答得很慢怎么办？**
A：如果你配置了自定义API Key，响应速度取决于你所选AI服务商。使用默认配置时，如果网络较慢，可能需要等待几秒。
